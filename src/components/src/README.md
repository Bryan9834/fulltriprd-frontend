FullTripRD – Booking Platform

Production-ready booking platform built with React and Vite, integrated with Supabase (PostgreSQL).
Implements authentication, role-based access control, protected routes, and structured booking lifecycle management through an admin dashboard.

🧱 Architecture Overview

This project represents the frontend layer of a full-stack booking system.

Client-side application built with React 19

Built and bundled using Vite

Integrated with Supabase (PostgreSQL)

Role-based authentication system

Protected routes for admin access

Booking lifecycle state management

Admin dashboard for reservation control

The application follows modular structure principles separating UI components, routing logic, and service-layer API communication.
